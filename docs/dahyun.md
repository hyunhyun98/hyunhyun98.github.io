# GIT HUB
## Day 1
> #git의 필요성# : 소스 코드 백업을 안전하게 할 수 있다.
>	> untracked : git이 변경사항을 추적하지 않는 파일
>	> modified : 파일을 변경했지만 아직 스텡이징 되지 않은 상태
>	> staged : 변경된 파일을 커밋이 될 리스트에 포함시킨 상태
>	> committed : git 데이터베이스에 하나의 버전으로 저장된상태
